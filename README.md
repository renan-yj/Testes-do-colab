## 📝 Descrição
Este repositório contém um projeto prático focado no aprendizado e aplicação de conceitos fundamentais de Data Science e Machine Learning. 
O desenvolvimento foi realizado integralmente via Google Colab, explorando a flexibilidade da nuvem para o processamento de dados e treinamento de modelos.

## 📌 Objetivo do Projeto
**O foco principal aqui é o aprimoramento técnico em:**
* **Análise Exploratória de Dados (EDA):** Entender padrões e correlações.
* **Pré-processamento:** Tratamento de dados ausentes, normalização e codificação.
* **Modelagem Preditiva:** Aplicação e avaliação de algoritmos de Machine Learning.
* **Cloud Development:** Uso eficiente do ecossistema Google Colab.

## 🛠️ Tecnologias e Bibliotecas
* **Linguagem:** Python
* **Manipulação de Dados:** Pandas, NumPy
* **Visualização:** Matplotlib, Seaborn
* **Machine Learning:** Scikit-Learn
* **Interface (Opcional):** Streamlit

## 📁 Estrutura do Repositório
* `acrobot/`: Pendulo duplo - objetivo de deixar ele reto.
* `cartpole/`: Carrinho com palito - objetivo - não deixar ele cair com os fatores externos (vento)
